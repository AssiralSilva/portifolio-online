Sobre mim

Olá, meu nome é Larissa Silva e sou Técnica em Informática para Internet e Desenvolvedora Front-End.

Faço uma formação pelo DevClub, para aprimorar conhecimentos. Fiz projetos pelo curso, como páginas principais em HTML5 e CSS3 e também, para colocar no portifólio, estou construindo um site de vendas de pipocas gourmet para uma amiga. A linguagem de código será JavaScript. Mas em breve farei ajustes com o framework React.

Procuro fazer sistemas Freelancers para aprimorar conhecimentos, ganhar experiência prática e expandir minha rede profissional.

O que significa isso?:

Faço a parte visual de um site, aquilo que conseguimos interagir. Quem trabalha com Front End é responsável por desenvolver por meio do código uma interface gráfica e, normalmente, com as tecnologias base da Web: HTML, CSS e JavaScript.
